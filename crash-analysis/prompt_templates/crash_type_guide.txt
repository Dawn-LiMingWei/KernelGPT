#### KASAN use-after-free 特征
- 报告常包含 Allocated/Freed 调用栈
- 重点对比分配点与释放点生命周期
- 常见原因：引用计数错误、并发释放、回调未取消

#### NULL pointer dereference 特征
- 访问地址接近 0 或明显无效
- 重点追踪指针初始化和错误路径
- 常见原因：缺少 NULL 检查、异常路径未初始化

#### Deadlock 特征
- 报告包含 lockdep 锁依赖链
- 重点寻找循环等待顺序
- 常见原因：锁顺序不一致、递归加锁

#### Bad access / paging request 特征
- 页面故障地址非法或权限冲突
- 重点检查地址计算和对象边界
- 常见原因：越界、悬空指针、错误偏移

#### Data-race 特征
- KCSAN 报告读写竞争位置
- 重点验证同步原语覆盖范围
- 常见原因：缺少锁、原子性不足、时序窗口
